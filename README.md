# SurFinity - Application

## Overview
This project was developed as part of the **PIDEV - 4th Year Engineering Program** at **Esprit School of Engineering** (Academic Year 2025-2026).

**SurFinity** is a management application designed to streamline the operations of insurance, credits, accounts, complaints, and news. Each module was developed collaboratively by our team as part of the PIDEV project.

## Features
- **User Management** (Nesrine)  
- **Insurance Management** (Tasnim)  
- **Credit Management** (Meriem)  
- **Account Management** (Molka)  
- **Complaints & News Management** (Yosr)  
- **SMS Notifications**  
- **Admin Dashboard**
